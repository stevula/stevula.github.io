Dev Bootcamp Cheat-Sheet: Centering in CSS

Centering things in CSS can be a pain in the ass. That's why I decided to make a handy-dandy cheat-sheet on the topic. These aren't the only ways to center things, but I think I've included the most common methods. I have encountered most of these techniques previously in my learning, but I would like to credit CSS-Tricks, Vanseo Design, and the blog of David Walsh for teaching me a couple new ones and clarifying the limitations of each technique for me.

    When attempting to center an element, consider these parameters. I'm only going to cover the first four for the purposes of this cheat-sheet.

  • Horizontal vs vertical alignment
  • Inline vs block element
  • Static/relative vs fixed/absolute positioning
  • Known vs unknown/dynamic height
  • Single vs multiple elements

Horizontal Alignment

Horizontal centering tends to be the simpler problem.

Text and other inline elements:

set parent element: text-align: center;

Block elements (static/relative):

This only works when the object has a declared width.

margin: 0 auto;

Block elements (fixed/absolute):

left: 50%; margin-left: (-50% of element width);

Vertical Alignment

In contrast to horizontal centering, vertical centering is evil incarnate. Counterintuitively, the vertical-align property is rarely the appropriate choice.

A single line of text:

set parent element: line-height: (parent element height);

Text and other inline elements

set parent element: padding: 50% 0;

Simulate a table (tables cells have simpler vertical alignment):

set parent element: display: table;
set child element: display: table-cell; vertical-align: middle;

Flexbox for inline elements:

set parent element: display: flex; align-items: center;

Block elements with known height:

top: 50%; margin-top: (-50% of element width);

Block elements with unknown/dynamic height:

position: absolute; top: 50%; transform: translateY(-50%);

Flexbox for block elements:

set parent element: display: flex; flex-direction: column; justify-content: center;
